## 🔍 测试方法

每完成一个功能后，必须测试：

### 测试步骤
1. 运行构建：`cd apps/web && npm run build`
2. 启动服务器：`./start-pwa.bat`
3. 访问移动端：`http://100.94.241.53:8000/m/`
4. 测试对应功能（见各文档的测试清单）

### 测试通过标准
- ✅ 点击按钮有反应（不是404或空白）
- ✅ 能看到实际内容（不是"即将推出"）
- ✅ 能完成完整流程（不卡在中间）
- ✅ 控制台无错误（F12查看）

## 🚨 常见错误和解决

### 错误1：TypeScript 类型错误
**解决：** 查看文档中的类型定义，完全复制

### 错误2：找不到文件
**解决：** 检查路径是否正确，使用 `@/` 开头的绝对路径

### 错误3：按钮点击无反应
**解决：** 检查是否实现了 onClick 处理函数

### 错误4：API 调用失败
**解决：** 检查后端是否启动，URL 是否正确
