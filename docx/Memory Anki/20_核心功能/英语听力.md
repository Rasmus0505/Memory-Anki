# 英语听力

## 背景

英语听力模块聚焦沉浸拼写、句子播放暂停逻辑、翻译区同步、视频布局和柔和音效反馈。

## 目标体验

- 页面不需要缩放或滚动即可同时看到视频、拼写区和翻译区。
- 视频按比例显示，不拉伸，不出现大量无意义黑边。
- 拼写区简单、显眼、低干扰。
- 音效柔和可调，不影响长期训练。

## 播放逻辑

- 播放完一句后暂停，等待用户完成本句拼写。
- 完成拼写包括手动输入正确，或通过快捷键提示使全部单词显示。
- 本句全部单词出现后，立即将翻译区更新为本句翻译。
- 完成后自动重播本句，并无停顿衔接播放下一句，然后自动暂停。
- 翻译区在未完成前保持上一句翻译，避免提前泄题。

## 布局与拼写

- 视频面积应扩大，同时保证拼写区完整可见。
- 删除不必要文字和说明，为视频与拼写区节省空间。
- 拼写区从字母方框改为下划线形式，单词间留适当间隔。
- 长句可以适度压缩视频，但不要频繁改变视频面积，避免视觉跳动。

## 音效反馈

- 支持键盘打字音、答对/错单词提示音、完成句子音。
- 音效柔和，可调音量，并参考脑图反馈音量体系。
- 拼写中字母颜色变化要清楚但不刺眼。

## 验收标准

- 一屏内能完成主要训练，不需要上下滑动。
- 长句不会让拼写区溢出或遮挡。
- 播放/暂停/重播/下一句衔接符合训练节奏。
