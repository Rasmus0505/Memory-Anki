# 计时器与专注

## 背景

计时器不仅是页面内计时组件，还承担专注提醒、休息召回、跨场景时间记录、反馈激励和全局悬浮显示。

## 目标体验

- 无论用户处于哪个网站页面，计时器都可见。
- 用户切换场景或短暂离开时，一级/二级计时显示不中断。
- 数据入库时按场景记录，但前台计时体验保持连续。
- 休息结束或返回学习场景时，系统能自动切回学习计时。

## 功能需求

- 全局悬浮计时器支持拖拽、调整大小、折叠为胶囊。
- 胶囊形态也可拖动，并能稳定恢复为卡片。
- 支持一级目标时间和二级小间隔。
- 最显眼数字显示二级时间，一级累计进度放在下方。
- 一级时间显示为 `累计时间/目标时长 倍率`，累计时间使用正计时。
- 退出页面或离开场景后，在闲置阈值内返回应合并为同一条记录。
- 随心模式、做题、复习、练习、编辑都应被识别为学习状态。

## 反馈要求

- 每次完成二级时间触发即时视觉反馈和音效。
- 二级完成次数越多，反馈越强。
- 达成一级目标时触发最大强度反馈。
- 反馈系统应复用统一反馈能力，不在计时器里重复实现。

## 桌面级悬浮讨论

如果要在网站外、电脑任意页面都可见，需要从网页内悬浮升级为桌面应用或系统级窗口能力。该能力属于后续架构扩展，不应和当前网页内全局悬浮混为一谈。

## 验收标准

- 页面切换不重置计时显示。
- 闲置阈值内返回能合并同一条时间记录。
- 胶囊可拖动、可恢复。
- 思维导图半屏或全屏时计时器仍显示在最高层级。
