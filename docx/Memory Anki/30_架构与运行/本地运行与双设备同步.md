# 本地运行与双设备同步

## 背景

Memory Anki 是个人自用、本地优先、电脑端使用的学习产品。当前通过百度网盘同步空间在两台电脑之间同步数据，同时希望生产开发环境和稳定学习环境分离。

## 目标体验

- 日常学习使用稳定版，不受后台开发影响。
- 开发环境可以频繁改代码、热更新、测试新功能。
- 两套环境共享或可控访问同一份学习数据，避免学习进度割裂。
- 启动方式稳定，不因代码更新频繁损坏 `start.bat`。

## 运行要求

- 后台改代码不影响用户正常使用网站。
- 后台改代码不影响记忆宫殿、时间记录、题库等数据保存。
- 网页刷新后能切换到新版本，不需要用户理解复杂启动流程。
- 数据库加载要快，不能使用时长时间空白。

## 生产/稳定版分离

- 推荐将稳定使用目录和开发 worktree 分开。
- 数据库应从代码目录中解耦，放到受控的数据目录。
- 开发环境直接写共享数据库前，应明确备份、锁定、迁移和回滚策略。
- 如存在破坏性迁移，应先在开发数据副本验证。

## 双设备同步

- 当前假设使用百度网盘同步空间。
- 同步重点是学习数据，而不是开发构建产物。
- 未来需要更清楚的冲突处理和数据版本说明，避免人工翻文件夹解决冲突。

## 验收标准

- 稳定版和开发版可以独立启动。
- 开发版崩溃不影响稳定版学习。
- 数据目录位置清楚，备份和迁移策略清楚。
