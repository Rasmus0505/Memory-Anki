## 当前最重要的产品方向

- Memory Anki 是本地优先的个人学习训练系统，不是普通笔记库。
- 核心闭环是：学习材料 -> 结构化 -> 记忆宫殿 -> 题库 -> 复习 / 练习 -> 反馈 -> 下一步训练。
- 优先级最高的体验是低负担进入训练、不中断心流、清楚保存进度、强反馈。
- AI 应作为可追踪、可配置、可确认的助教，避免黑箱调用昂贵模型。
