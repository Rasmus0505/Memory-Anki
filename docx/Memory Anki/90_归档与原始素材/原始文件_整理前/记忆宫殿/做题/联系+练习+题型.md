# 联系+练习+题型

> 原始路径：记忆宫殿\做题\联系+练习+题型.md

如何实现复习一个章节的思维导图的时候，除了我现在已经在做的：主动回忆代替重读（提升记忆效率），我想新增一个AI帮忙的功能：练习+联系，我希望在复习思维导图的时候，单纯的记忆活动让我疲惫，可以顺手生成一些题目让我玩玩，这些题目可以是本章节的也可以是本章节（加强本章节练习）和其他宫殿内容相关联（强化知识间的关联），对于好的题目可以加入题库，或者即做即丢，额外提到一个题库功能就算题目要标记来源：复习的时候生成的还是普通的题目（比如pdf ai转题目这类），方便我判断题目来源

初步构思：qwen flash负责，如果是本章节内容则将本章节完整思维导图内容发给flash然后出本章的出题，如果选择和其他宫殿相联系则将本章节完整思维导图+其他章节的第一层多节点内容作为背景发给AI然后出本章和其他宫殿相关联的题，第一层多节点举例：1-1-1这种即使有了三个节点但也不算第一层多节点，最后一层必须大于等于两个节点，可以选项框直接选择题目类型+题目数量，选择题点击答案后直接判错+显示解析无需手动点击
直接放进复习页面上方的工具栏吧，命名“做题休息”，我点击“做题休息”，然后进行配置，再生成，
选择“和其他宫殿相联系”时，其他宫殿默认全选，可额外点击开一个弹窗专门手动调整选择，按照层级划分，包括：学科/宫殿，题型选择默认支持选择题 + 判断题 + 填空题 + 连线题 + 排序题 + 归类题 + 简答题

- 选择题：点击选项后立即判错，正确项变绿，错误项变红，解析自动展开。
- 判断题：两个大按钮 对 / 错，点击后立即反馈；如果错，显示“错误点在哪里”。
- 填空题：一句话中挖 1-3 个空，每个空输入后点enter直接快速提交该空；然后有个总提交按钮可以直接提交全部
- 连线题：左右两列卡片，点击左边再点右边完成连线；提交后错线标红、对线标绿。注意通过颜色表示用户选择的卡片让用户少感到疑惑
- 排序题：卡片拖拽排序，也提供上下移动按钮；提交后显示正确顺序。
- 归类题：上方是待分类知识点，下方是分类桶，拖进去后提交；错放的卡片高亮。
- 简答题：输入答案后先显示参考答案和解析；可额外点 AI点评我的答案。
