## 一句话愿景

Memory Anki 未来不只是“记忆宫殿 + Anki + AI 生成题”的集合，而是一个本地优先的个人知识训练系统：

```text
材料进入系统
  -> AI 结构化
  -> 形成知识大纲 / 记忆宫殿 / 题库
  -> 进入今日训练台
  -> 完成复习、练习、做题、阅读、听力
  -> 产生学习事件和反馈信号
  -> 系统推荐下一步
  -> 持续修正知识结构和训练计划
```

最终目标：

- 打开应用，不需要想“我该去哪一个页面”。
- 系统直接告诉我“现在最值得做什么，为什么，预计多久”。
- 做完之后马上获得反馈：掌握了什么、哪里薄弱、下一步是什么。
- AI 像助教一样提出建议、生成草稿、解释原因，关键写入前由我确认。
- 所有学习行为都沉淀为可追踪事件，长期形成个人能力画像和训练策略。

## 最终目标画面

理想中的一天：

1. 打开 Memory Anki。
2. 首页显示“今日训练台”：
   - 先做 6 分钟英语热身。
   - 然后清理 3 个逾期复习。
   - 再做 12 道错题。
   - 最后处理一个 AI 生成题库草稿。
3. 点击“开始 20 分钟训练”。
4. 系统自动进入心流训练空间。
5. 做题、复习、红标节点、英语任务连续穿插。
6. 结束后显示：
   - 完成了什么。
   - 哪些知识点仍薄弱。
   - 哪个宫殿需要补题。
   - 明天优先做什么。
7. AI 助教提示：
   - “这个章节错题集中在概念辨析，建议补 5 道判断题。”
   - “这个宫殿第 3 段节点过长，建议拆成 mini-palace。”
   - “这篇阅读材料偏难，下一篇建议降半级。”

> 未来的 Memory Anki：把知识材料变成训练内容，把训练行为变成反馈信号，再把反馈信号变成下一步行动。
