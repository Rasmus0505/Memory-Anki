## 🔄 与桌面端的关系

### 代码隔离
```
✅ 移动端代码独立在 features/mobile/
✅ 不影响桌面端代码
✅ 共享 shared/ 工具函数
✅ 复用数据模型和 API
```

### 路由隔离
```
桌面端：/  /dashboard  /palaces  /review  /english  /profile
移动端：   /m/freestyle /m/palaces /m/review /m/english /m/profile
```

### 启动方式
| 方式 | 桌面端 | 移动端 |
|------|--------|--------|
| 路径访问 | `/` | `/m/*` |
| Standalone PWA | ❌ | ✅ |
| 浏览器 | ✅ | ✅ |
