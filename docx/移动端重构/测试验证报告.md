## 🔧 修复的问题

### 问题 1：类型错误
**错误信息：**
```
FreestyleProgressSnapshot 类型不匹配
- 使用了不存在的 seed 属性
- 使用了不存在的 completedIndices 属性
```

**解决方案：**
查看了正确的类型定义：
```typescript
interface FreestyleProgressSnapshot {
  currentIndex: number
  questionStates: Record<number, QuizRuntimeState>
  correctStreak: number
  resolvedQuestionIds: number[]
  lastQueueSignature: string
}
```

**修复内容：**
1. 移除了 `nextFreestyleSeed` 的导入
2. 修改 `handleResetProgress` 使用正确的属性
3. 将 `completedIndices` 改为 `resolvedQuestionIds`

**文件：** `apps/web/src/features/mobile/pages/freestyle/MobileFreestylePage.tsx`
