## 🏗️ 技术架构

### 组件层次
```
MobileApp (根组件)
├── MobileAppRouter (路由)
│   ├── MobileFreestylePage
│   ├── MobilePalaceListPage
│   ├── MobileReviewPage
│   ├── MobileEnglishPage
│   └── MobileProfilePage
└── MobileTabBar (底部导航)
```

### 路由设计
| 路由 | 页面 | 功能 |
|------|------|------|
| `/m/` | → `/m/freestyle` | 重定向 |
| `/m/freestyle` | MobileFreestylePage | 随心模式 |
| `/m/palaces` | MobilePalaceListPage | 宫殿列表 |
| `/m/review` | MobileReviewPage | 复习系统 |
| `/m/english` | MobileEnglishPage | 英语学习 |
| `/m/profile` | MobileProfilePage | 个人设置 |

### 技术栈
- **前端框架：** React 18 + TypeScript
- **路由：** React Router v6
- **样式：** Tailwind CSS + 自定义 CSS
- **图标：** Lucide React
- **构建：** Vite
- **PWA：** Workbox (Service Worker)

## 🔄 与桌面端的关系

### 完全隔离
```
桌面端：apps/web/src/features/* (除 mobile 外)
移动端：apps/web/src/features/mobile/*
共享：  apps/web/src/shared/*
```

### 数据共享
- ✅ 复用数据模型（如 `freestyle/model/`）
- ✅ 复用 API 调用
- ✅ 共享 localStorage 数据
- ✅ 同一后端服务

### 路由隔离
- **桌面端：** `/` `/dashboard` `/palaces` 等
- **移动端：** `/m/*` 路径
- **互不干扰** - 各自独立运行
