
## 需求1 — 修复导图 Ctrl 多选失效

### 原始输入
用户反馈思维导图里按住 Ctrl 不能连续多选卡片，这个功能会反复失效，希望稳定恢复。

## 需求2 — 提升图片转脑图的 JSON 容错

### 原始输入
用户反馈图片转脑图在导入侧栏里报“模型返回内容不是有效的脑图 JSON”，希望修好这条识别链路。

### plan需求输入
修复图片转脑图对模型输出 JSON 的脆弱解析。保留转脑图模式，不自动降级到转文字；当解析失败时显示友好错误，并附简短的模型返回摘要，帮助判断是模型输出包裹代码块、带前后缀，还是图片本身结构不适合直接转脑图。

## 需求3 — 仪表盘中部两张卡片改版

### 原始输入
用户希望把仪表盘中部左卡片从“今日到期”改成展示今天每个宫殿的学习时长，用一条分色进度条表示编辑、练习、复习的占比，并且悬停时能看到具体时间；右卡片从“最近编辑”改成“新增章节数量：x”，下面列出今天新增的记忆宫殿，并按章节和小节的层级关系排版。

### plan需求输入
仪表盘中部左卡片改为“今日学习”，按宫殿汇总当天编辑、练习、复习三类时长，并显示总时长和分色占比条；右卡片改为“新增章节数量：x”，x 按今天新增记忆宫殿数量统计，列表展示今天新增的宫殿并按章节、小节层级分组显示，未关联章节的单独归类。
