## 原始输入

## 最终输入

### Bug 修复：保存后层级丢失
- edit 页面保存时桩子的树形结构不能丢——当前因为新桩子无 id 导致子节点 parent_id 丢失变成根节点

### 统一节点模型
- 不再区分"桩子"和"记忆内容"，使用统一节点
- 每个节点：名称（可选）+ 内容（可选）
- 可以只填名称、只填内容、或都填
- 任意节点都可以挂任意子节点，形成自由树形结构

### 拖拽移动
- 左侧六个点图标（GripVertical）作为拖拽手柄
- 可拖拽节点到同级或嵌套到其他节点下

### 中文化
- 记忆宫殿编辑页面全部文本改为中文
