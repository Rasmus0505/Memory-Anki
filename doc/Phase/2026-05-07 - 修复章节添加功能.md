# 修复章节添加功能

## 原始输入

无法添加根章节和子章节，修复bug

## 最终输入

### 根章节添加失败（前端 React Hooks 错误）

- 选择学科后输入章节名点击添加，页面报错：`React has detected a change in the order of Hooks called by TreeRenderer`
- 错误发生在 `renderChapterNode` 中——该函数在 Knowledge 组件体内定义，作为 render prop 传入 TreeRenderer
- 当 `editingChapter` 状态变化时，renderChapterNode 返回的 JSX 结构发生变化（条件渲染 Input/Button），导致 hooks 顺序不一致

### 子章节添加失败（后端 HTTP 500）

- 在已有章节上点击 + 添加子章节，后端 `POST /subjects/{subject_id}/chapters` 返回 500
- 后端 `create_chapter` 端点缺少异常处理，无法定位具体错误原因
- Chapter 模型自引用关系 `children`/`parent` 的 `remote_side` 配置疑有误（两侧 `remote_side` 值互换）
