# 用户需求记录

## 2026-05-06 — 项目初始化

### 核心需求
- 记录每天创造的记忆宫殿
- 按照记忆曲线自动安排复习
- 交互舒服自然，高度可定制

### 技术选型
- Python FastAPI 后端 + React (shadcn/ui) 前端
- 前后端分离架构
- SQLite 本地数据库
- 文件附件存储

### 功能要求
1. **记忆宫殿管理**：标题、描述（Markdown）、记忆桩（位置+内容）、标签/分类、难度评级(1-5)、图片/附件（所有字段可选填）
2. **复习模式**：混合模式 — 闪卡式问答 + 浏览式自由回忆，可在宫殿级别配置
3. **记忆曲线算法**：三种可选
   - 艾宾浩斯标准间隔 (1,2,4,7,15,30,60 天)
   - SM-2 动态调整（根据评分动态改变间隔）
   - 完全自定义间隔
4. **配置中心**：所有参数通过 UI 配置，切换算法对历史数据可选择性生效
5. **导入导出**：支持 JSON（备份恢复）和 Markdown（便携阅读）双向导入导出
6. **仪表盘**：今日待复习统计、本周完成率、近期活跃宫殿

### 交互要求
- 闪卡模式：看到标题 → 点击翻开 → 展示全部内容+记忆桩 → 自评 0-5 分 → 自动安排下次复习
- 浏览模式：直接展示完整内容 → 标记已复习 → 下一项
- 复习模式默认 flashcard

### 项目规范
- 项目入口在根目录，方便一键打开
- doc/ 记录用户需求
- 根目录 CLAUDE.md 为 AI 协作提示词
