# 07-08 收敛计时器样式

## 1. 原始需求

`apps/web/src/index.css` 中存在两套几乎平行的计时器 UI 样式（实测行段，与任务原描述基本一致）：

- **`.memory-anki-global-timer-*`**：第 900~1229 行（约 330 行），网页内悬浮计时器（`GlobalTimerProvider` 渲染），含 panel/dragbar/digits/row/actions/custom-break/capsule/resize 手柄/screen-pulse。
- **`.memory-anki-timer-overlay-*`**：第 1302~1557 行（约 255 行），Electron 桌面悬浮窗（`TimerOverlayApp`/`TimerOverlayPage` 渲染，`apps/desktop-timer/main.cjs` 加载 `/timer-overlay` 路由），含 shell/header/digits/copy/actions/minutes-input/capsule/dot，另有 `-webkit-app-region: drag` 等 Electron 专属属性。

两套之间夹着 break-guard 样式（1231~1300 行，共用 `memory-anki-break-guard-card-alert` 动画，不在本文档范围）。

**核实修正**：任务原描述提到的"第三套 SessionTimerBar 样式"实际上**不存在于 index.css**——`shared/components/session/SessionTimerBar.tsx` 全部使用 Tailwind 工具类（`rounded-lg border border-border/70 bg-background/95 ...`），没有专属 CSS 段。因此本文档只合并两套。

逐块对照出的平行重复（选择器不同、声明高度雷同）：

| 语义 | global-timer（行号） | timer-overlay（行号） | 差异点 |
|---|---|---|---|
| 容器底色/边框/阴影 | `-panel` 911~927 + `-panel-break` 929 + `-panel-expired` 937 | `-shell` 1302~1320 + `-shell-break` 1322 + `-shell-expired` 1330 | 圆角 28px vs 22px、padding 变量 vs clamp、渐变透明度 ±0.02~0.06 |
| 大数字 | `-digits` 998~1008（64px、900 字重） | `-digits` 1393~1404（clamp 32~62px、900 字重） | 尺寸策略（固定+变量 vs clamp） |
| 胶囊态 | `-capsule` 1189~1206 + `-capsule-dot` 1208 + `-capsule-label` 1216 | `-capsule` 1455~1474 + `-dot` 1508~1532 + `-capsule-label` 1481 | overlay 多状态色 dot（running/break/expired）与 app-region |
| 自定义休息输入 | `-custom-break` 1052 + `-custom-input` 1077 + `-custom-submit` 1085 | `-custom-break` 1430 + `-minutes-input` 1447 | 尺寸 |
| 操作按钮行 | `-actions` 1045 + `-action-button` 1069 | `-actions` 1423 + `-actions > button` 1437 | 尺寸 |
| 到期告警动画 | 943 引用 `memory-anki-break-guard-card-alert` | 1336、1478 同一动画 | 已共用 |

目标：抽出公共 CSS 自定义属性与共享基类，删除重复声明（预估净减 80~120 行），且两处 UI 视觉零回归。

### 步骤 2：抽公共 CSS 变量（新增，不删任何行）

在 index.css 的 `:root`（第 332 行 `--memory-anki-feedback-ease` 所在块）追加计时器公共变量：

```css
:root {
  /* 计时器公共外观 */
  --timer-surface-border: rgba(148, 163, 184, 0.26);
  --timer-surface-bg: rgba(255, 255, 255, 0.95);
  --timer-surface-shadow: 0 0 0 1px rgba(255, 255, 255, 0.44), 0 24px 60px rgba(15, 23, 42, 0.18);
  --timer-break-border: rgba(251, 146, 60, 0.36);
  --timer-expired-border: rgba(248, 113, 113, 0.5);
  --timer-digits-color: #111827;
  --timer-digits-weight: 900;
}
```

- 数值取两套现值的公共部分；两套原有细微差（如阴影扩散 70px vs 60px）统一到其一，属可接受的视觉收敛。
- **自查点**：仅新增变量，`npm run build` 通过，页面无变化。

### 步骤 3：逐块替换为变量/共享类（每块一个小改动）

按第 1 节表格逐行替换，示例（容器块）：

修改前（`-panel` 911~927 与 `-shell` 1302~1320 各自写死）：

```css
.memory-anki-global-timer-panel {
  border: 1px solid rgba(148, 163, 184, 0.26);
  /* ... */
}
.memory-anki-timer-overlay-shell {
  border: 1px solid rgba(148, 163, 184, 0.42);
  /* ... */
}
```

修改后（引用变量；overlay 的 0.42 边框透明度并入 0.26——桌面窗口有系统边界，视觉差可忽略；若人工验收觉得太淡，给 shell 单独保留一行覆盖）：

```css
.memory-anki-global-timer-panel,
.memory-anki-timer-overlay-shell {
  border: 1px solid var(--timer-surface-border);
}
```

依次处理：break/expired 变体边框与渐变、digits 颜色与字重、capsule 底色/边框/阴影、dot 基础尺寸（10px 圆点）、actions/input 的背景色。**保留**各自独有声明：overlay 的 `clamp()` 尺寸、`-webkit-app-region`、grid 行模板；global-timer 的 resize 手柄全部（1090~1187 行，overlay 无对应物，不动）、`--timer-panel-*` 尺寸变量体系。

- 一次只合并一个语义块，改完立即在浏览器目视对比。
- **不要**合并 break-guard 段（1231~1300）；**不要**动 keyframes（1559~1617）；**不要**重命名任何 tsx 中正在使用的类名（本文档只动 CSS 声明，不动类名与 DOM）。
- **自查点**：每块替换后 `rg -c "memory-anki-global-timer|memory-anki-timer-overlay" src/index.css` 的选择器数量不减少（类名都还在），只有声明行数减少。
