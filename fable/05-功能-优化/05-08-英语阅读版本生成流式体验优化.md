# 05-08 英语阅读版本生成流式体验优化

## 1. 原始需求

英语阅读版本生成耗时较长，单次阻塞式返回会让前端缺少进度反馈。目标是在保留原有同步接口的同时，为 `apps/api/src/memory_anki/modules/english_reading` 增加 SSE 流式生成事件。

## 2. 详细执行清单

1. 在 `apps/api/src/memory_anki/modules/english_reading/application/version_service.py` 提供 `generate_material_version_events`。
2. 在 `apps/api/src/memory_anki/modules/english_reading/presentation/router.py` 增加 `/english-reading/materials/{material_id}/generate/stream`，使用 `StreamingResponse` 与 `text/event-stream`。
3. 保持原 `/generate` 同步接口兼容。

## 3. 测试验收标准

- `cd apps/api && python -m pytest tests/test_english_reading_routes.py -q` 通过。
- 测试覆盖响应头包含 `text/event-stream`。
- 回归要求：同步生成、材料创建、词典查询不能被破坏。
