# 08-17 字体资源本地化（去 Google Fonts CDN）

## 1. 原始需求

前端 HTML 和离线页不应依赖 Google Fonts CDN，否则离线/PWA 和国内网络环境会受影响。目标删除外链字体，并改为系统字体栈。

## 2. 详细执行清单

1. 修改 `apps/web/index.html`，移除 Google Fonts preconnect/stylesheet。
2. 修改 `apps/web/src/index.css`/`apps/web/src/styles/foundation.css` 字体变量为系统字体栈。
3. 修改 `apps/web/public/offline.html`，移除 `Inter` 假设。

## 3. 测试验收标准

- `rg -n "fonts\\.googleapis|fonts\\.gstatic|https://fonts|Inter|JetBrains Mono" apps/web/index.html apps/web/src/index.css apps/web/src/styles apps/web/public/offline.html` 无命中。
- `cd apps/web && npm run typecheck` 通过。
