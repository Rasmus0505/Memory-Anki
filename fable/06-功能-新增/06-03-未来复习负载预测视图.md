# 06-03 未来复习负载预测视图

## 1. 原始需求

用户完全看不到"未来几天要还多少复习债"：

1. `review_schedules` 表（`apps/api/src/memory_anki/infrastructure/db/_tables/palaces.py` 第 259-292 行）里每条未完成日程都有 `scheduled_date`，且已有复合索引 `ix_review_schedules_due_lookup (completed, scheduled_date, scheduled_at, id)`，按日聚合是覆盖索引查询。
2. 复习总览页 `apps/web/src/app/router/review/ReviewOverview.tsx` 只显示"今天到期 N 项 / 逾期 N 项"（来自 `GET /api/v1/review/queue` 的 `due_count`/`overdue_count`），新建宫殿或平滑逾期后，用户无法预判未来一周是否会堆成山。
3. 前端已有 recharts（`apps/web/package.json`：`recharts: ^3.8.1`），dashboard 已用它画图，复习页加一张柱状图零新依赖。

期望效果：复习总览页新增"未来负载"卡片，默认展示未来 7 天每天到期数量的柱状图（含"逾期堆积"一柱），可切换 30 天。

## 2. 详细执行清单

> 硬性约束：**不要**新建表、**不要**写 alembic 迁移；**不要**改 `review_queue_service.py` 的排队逻辑（本功能只读聚合）；**不要**在 `GET /review/queue` 里顺带塞预测数据（保持端点职责单一，图表懒加载）。

#### 步骤 1：在 reviews 模块新增聚合函数

打开 `apps/api/src/memory_anki/modules/reviews/application/review_metrics_service.py`（83 行，已有 `get_weekly_stats` 等聚合函数），在文件末尾追加：

```python
def get_review_load_forecast(session: Session, days: int = 7) -> dict:
    """未来 days 天每日到期复习量 + 逾期堆积量（只统计未完成、未归档、未掌握）。"""
    from sqlalchemy import func

    from memory_anki.infrastructure.db.models import Palace, ReviewSchedule

    days = max(1, min(int(days), 60))
    today = date.today()
    end = today + timedelta(days=days - 1)

    rows = (
        session.query(
            ReviewSchedule.scheduled_date,
            func.count(ReviewSchedule.id),
        )
        .join(Palace, ReviewSchedule.palace_id == Palace.id)
        .filter(
            ReviewSchedule.completed == False,  # noqa: E712
            Palace.archived == False,  # noqa: E712
            Palace.mastered == False,  # noqa: E712
            ReviewSchedule.scheduled_date <= end,
        )
        .group_by(ReviewSchedule.scheduled_date)
        .all()
    )

    overdue_count = 0
    by_date: dict[str, int] = {}
    for scheduled_date, count in rows:
        if scheduled_date is None:
            continue
        if scheduled_date < today:
            overdue_count += int(count)
        else:
            by_date[scheduled_date.isoformat()] = int(count)

    items = []
    for offset in range(days):
        day = today + timedelta(days=offset)
        key = day.isoformat()
        items.append(
            {
                "date": key,
                "due_count": by_date.get(key, 0),
                "is_today": offset == 0,
            }
        )

    return {
        "days": days,
        "overdue_count": overdue_count,
        "total_upcoming": sum(item["due_count"] for item in items),
        "items": items,
    }
```

说明：文件顶部已 `from datetime import date, datetime, timedelta`，无需新增 import；`Palace`/`ReviewSchedule` 的局部 import 与该文件既有风格（顶部只 import 了 `ReviewLog`）保持一致也可提到顶部，二选一，别两处都写。过滤条件与 `freestyle/application/feed_service.py` 第 172-176 行对"有效复习"的定义一致（未完成 + 未归档 + 未掌握）。

#### 步骤 4：负载预测卡片组件（FSD：features/review/components）

新建 `apps/web/src/features/review/components/ReviewLoadForecastCard.tsx`：

```tsx
import { useEffect, useState } from 'react'
import { Bar, BarChart, CartesianGrid, ResponsiveContainer, Tooltip, XAxis, YAxis, Cell } from 'recharts'
import { Card, CardContent, CardHeader, CardTitle } from '@/shared/components/ui/card'
import { Button } from '@/shared/components/ui/button'
import { getReviewLoadForecastApi } from '../api'
import type { ReviewLoadForecastResponse } from '@/shared/api/contracts'

export function ReviewLoadForecastCard() {
  const [days, setDays] = useState<7 | 30>(7)
  const [data, setData] = useState<ReviewLoadForecastResponse | null>(null)

  useEffect(() => {
    let cancelled = false
    getReviewLoadForecastApi(days)
      .then((payload) => {
        if (!cancelled) setData(payload)
      })
      .catch(() => {})
    return () => {
      cancelled = true
    }
  }, [days])

  if (!data) return null
  const chartData = [
    ...(data.overdue_count > 0
      ? [{ date: '逾期', due_count: data.overdue_count, is_today: false, overdue: true }]
      : []),
    ...data.items.map((item) => ({
      ...item,
      overdue: false,
      date: item.is_today ? '今天' : item.date.slice(5),
    })),
  ]

  return (
    <Card>
      <CardHeader className="flex flex-row items-center justify-between">
        <CardTitle className="text-base">
          未来负载 · {days} 天共 {data.total_upcoming} 项
          {data.overdue_count > 0 ? `（另有 ${data.overdue_count} 项逾期）` : ''}
        </CardTitle>
        <div className="flex gap-1">
          <Button size="sm" variant={days === 7 ? 'default' : 'outline'} onClick={() => setDays(7)}>7 天</Button>
          <Button size="sm" variant={days === 30 ? 'default' : 'outline'} onClick={() => setDays(30)}>30 天</Button>
        </div>
      </CardHeader>
      <CardContent className="h-48">
        <ResponsiveContainer width="100%" height="100%">
          <BarChart data={chartData}>
            <CartesianGrid strokeDasharray="3 3" vertical={false} />
            <XAxis dataKey="date" fontSize={11} interval="preserveStartEnd" />
            <YAxis allowDecimals={false} fontSize={11} width={28} />
            <Tooltip formatter={(value) => [`${value} 项`, '到期复习']} />
            <Bar dataKey="due_count" radius={[3, 3, 0, 0]}>
              {chartData.map((entry) => (
                <Cell
                  key={entry.date}
                  fill={entry.overdue ? '#ef4444' : entry.is_today ? '#f59e0b' : '#6366f1'}
                />
              ))}
            </Bar>
          </BarChart>
        </ResponsiveContainer>
      </CardContent>
    </Card>
  )
}
```

`Card`/`Button` 的实际 import 路径以仓库中 `@/shared/components/ui/` 现有文件为准（DashboardPage 已在用）。
