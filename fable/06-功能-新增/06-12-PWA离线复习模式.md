# 06-12 PWA 离线复习模式

## 1. 原始需求

PWA 离线时应至少能打开一个复习入口，并对 feed API 给出缓存或明确空数据。目标是用 service worker contract 测试锁定 `/freestyle` 作为离线复习 shell。

## 2. 详细执行清单

1. 审计 `apps/web/public/sw.js` 的 precache、navigation fallback 和 API cache 策略。
2. 在 `apps/web/src/pwa/serviceWorkerContract.test.ts` 增加可执行 service worker harness。
3. 覆盖离线导航 `/review/session/42` 回退 `/freestyle` shell。
4. 覆盖 `/api/v1/freestyle/feed` 离线缓存与空 JSON fallback。

## 3. 测试验收标准

- `cd apps/web && npm test -- --run src/pwa/serviceWorkerContract.test.ts src/shared/persistence/mutationQueue.test.ts` 通过。
- 回归要求：PWA 更新提示 toast 不被破坏。
