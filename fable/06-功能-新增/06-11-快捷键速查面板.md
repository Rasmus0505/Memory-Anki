# 06-11 快捷键速查面板

## 1. 原始需求

全局命令面板已有快捷键入口，但缺少按 `?` 打开的速查视图。目标是在命令面板中展示真实注册的快捷键，避免文案和实际行为脱节。

## 2. 详细执行清单

1. 修改 `apps/web/src/app/shell/GlobalCommandPalette.tsx`，读取 `MEMORY_ANKI_SHORTCUT_ACTIONS`。
2. 使用 `readMemoryAnkiShortcuts` 与 `getShortcutLabel` 渲染快捷键组。
3. 监听 `?` 打开命令面板并显示快捷键组，不触发远程搜索。
4. 补 `apps/web/src/app/shell/GlobalCommandPalette.test.tsx`。

## 3. 测试验收标准

- `cd apps/web && npm exec vitest run src/app/shell/GlobalCommandPalette.test.tsx src/shared/i18n/messages.test.ts src/pwa/registerServiceWorker.test.ts src/pwa/serviceWorkerContract.test.ts` 通过。
- `cd apps/web && npm run typecheck` 通过。
