# 06-04 错题本与弱项重练

## 1. 原始需求

错题数据已经在积累，但没有任何消费入口：

1. `palace_quiz_questions` 表已有 `correct_count`/`incorrect_count`/`attempt_count` 三个计数列（`apps/api/src/memory_anki/infrastructure/db/_tables/palaces.py` 第 467-469 行），每次选择题作答都会更新（`POST /palace-quiz-questions/{question_id}/choice-attempts`，`modules/palace_quiz/presentation/router.py` 第 241 行）。
2. `freestyle_quiz_attempts` 表（同 `_tables/palaces.py` 第 492-530 行）记录了每次答题的 `question_id`、`is_correct`、`created_at`，且有 `ix_freestyle_quiz_attempts_question_created` 索引。
3. 随心模式 feed（`modules/freestyle/application/feed_service.py`）已支持 `range` 参数（`all`/`due`/`needs_practice`/`specific_palaces`），前端 `apps/web/src/features/freestyle/FreestylePage.tsx` 已有完整的刷题 UI 与答题上报链路——重练错题只需给 feed 加一个新 range，答题 UI 零改动。

期望效果：(a) 一个"错题本"聚合视图，按宫殿分组列出所有 `incorrect_count > 0` 的题、显示错误率与最近一次做错时间；(b) 一键"重练错题"，进入随心模式且 feed 只出错题。

**机制取舍**：复用 freestyle feed（本文档方案）优于新造 quiz 练习流——feed 的题卡（`quiz_cards.py`）自带 `serialize_question`、答题会写回 `freestyle_quiz_attempts` 和题目计数，形成"做错 → 进错题本 → 重练做对 → 计数更新"的闭环，不需要任何新会话状态。

#### 步骤 1：错题聚合服务

新建 `apps/api/src/memory_anki/modules/palace_quiz/application/wrong_questions_service.py`，完整内容：

```python
"""Aggregate wrong-question book from quiz counters and freestyle attempts."""

from __future__ import annotations

from sqlalchemy import func
from sqlalchemy.orm import Session, selectinload

from memory_anki.infrastructure.db.models import (
    FreestyleQuizAttempt,
    Palace,
    PalaceQuizQuestion,
)
from memory_anki.modules.palace_quiz.application.question_schema import serialize_question
from memory_anki.modules.palaces.application.title_sync_service import resolve_palace_title


def get_wrong_questions(session: Session, limit: int = 200) -> dict:
    limit = max(1, min(int(limit), 500))
    questions = (
        session.query(PalaceQuizQuestion)
        .options(
            selectinload(PalaceQuizQuestion.palace),
            selectinload(PalaceQuizQuestion.mini_palace),
        )
        .filter(PalaceQuizQuestion.incorrect_count > 0)
        .order_by(
            (PalaceQuizQuestion.incorrect_count * 1.0 / func.max(PalaceQuizQuestion.attempt_count, 1)).desc(),
            PalaceQuizQuestion.incorrect_count.desc(),
            PalaceQuizQuestion.id.asc(),
        )
        .limit(limit)
        .all()
    )

    question_ids = [question.id for question in questions]
    last_wrong_by_question: dict[int, str] = {}
    if question_ids:
        rows = (
            session.query(
                FreestyleQuizAttempt.question_id,
                func.max(FreestyleQuizAttempt.created_at),
            )
            .filter(
                FreestyleQuizAttempt.question_id.in_(question_ids),
                FreestyleQuizAttempt.is_correct == False,  # noqa: E712
            )
            .group_by(FreestyleQuizAttempt.question_id)
            .all()
        )
        for question_id, created_at in rows:
            if question_id is not None and created_at is not None:
                last_wrong_by_question[int(question_id)] = created_at.isoformat(timespec="minutes")

    items = []
    for question in questions:
        palace: Palace | None = question.palace
        items.append(
            {
                "question": serialize_question(question),
                "palace_id": question.palace_id,
                "palace_title": resolve_palace_title(palace) if palace else "",
                "incorrect_count": question.incorrect_count,
                "correct_count": question.correct_count,
                "attempt_count": question.attempt_count,
                "last_wrong_at": last_wrong_by_question.get(question.id),
            }
        )
    return {"total": len(items), "items": items}
```
