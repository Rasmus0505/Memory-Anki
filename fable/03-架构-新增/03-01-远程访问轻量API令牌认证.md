# 03-01 远程访问轻量 API 令牌认证

## 1. 原始需求

当前 `apps/api/src/memory_anki/app/main.py` 挂载的全部 `/api/v1` 路由**没有任何认证**，且第 143~149 行 CORS 配置为 `allow_origins=["*"]`。而 `configure-tailscale-pwa.bat` 会通过 `tools/pwa_launcher.ps1 ConfigureServe` 把 8012 端口经 Tailscale Serve 以 HTTPS 暴露给整个 tailnet，手机端 PWA 由此访问。虽然 Tailscale 网络本身有设备准入，但 tailnet 内任何设备（包括未来共享/借用的节点）都可以无凭据读写全部数据、下载附件、触发 AI 调用（消耗付费 API 额度）。

目标：增加一个**单令牌**的轻量校验中间件——环境变量 `MEMORY_ANKI_API_TOKEN` 配置令牌；远程请求必须携带令牌；本机（127.0.0.1 直连）请求豁免；未配置令牌时行为与现状完全一致（零配置默认不破坏任何现有流程）。**不引入** OAuth/JWT/用户体系/会话过期——这是单用户本地产品。

### 步骤 3：新建中间件文件 `core/api_token_auth.py`

新建文件 `apps/api/src/memory_anki/core/api_token_auth.py`，完整内容如下（风格参照既有 `apps/api/src/memory_anki/core/request_logging.py` 的 `BaseHTTPMiddleware` 写法）：

```python
from __future__ import annotations

import secrets

from fastapi import Request
from starlette.middleware.base import BaseHTTPMiddleware
from starlette.responses import JSONResponse
from starlette.types import ASGIApp

TOKEN_HEADER = "X-Memory-Anki-Token"
LOOPBACK_HOSTS = {"127.0.0.1", "::1", "localhost"}


def _extract_token(request: Request) -> str:
    custom = request.headers.get(TOKEN_HEADER, "").strip()
    if custom:
        return custom
    authorization = request.headers.get("Authorization", "").strip()
    if authorization.lower().startswith("bearer "):
        return authorization[7:].strip()
    return ""


def _is_direct_loopback(request: Request) -> bool:
    # Tailscale Serve 反代到 127.0.0.1 时会带 X-Forwarded-For，
    # 因此仅"回环地址 + 无转发头"才视为真正的本机请求。
    if request.headers.get("X-Forwarded-For"):
        return False
    client = request.client
    return client is not None and client.host in LOOPBACK_HOSTS


class ApiTokenAuthMiddleware(BaseHTTPMiddleware):
    """校验远程请求的 API 令牌；令牌未配置或本机直连时放行。"""

    def __init__(self, app: ASGIApp, token: str | None):
        super().__init__(app)
        self.token = (token or "").strip()

    async def dispatch(self, request: Request, call_next):
        if not self.token:
            return await call_next(request)
        path = request.url.path
        if not path.startswith("/api"):
            # PWA 静态资源不拦截，否则手机端连输入令牌的页面都打不开。
            return await call_next(request)
        if request.method == "OPTIONS":
            return await call_next(request)
        if _is_direct_loopback(request):
            return await call_next(request)
        provided = _extract_token(request)
        if provided and secrets.compare_digest(provided, self.token):
            return await call_next(request)
        return JSONResponse(
            status_code=401,
            content={"detail": "缺少或错误的 API 令牌（MEMORY_ANKI_API_TOKEN）。"},
        )
```

**自查点**：`cd apps/api && python -c "from memory_anki.core.api_token_auth import ApiTokenAuthMiddleware; print('ok')"` 输出 `ok`。

### 步骤 5：前端令牌存取模块 `shared/api/apiToken.ts`

新建文件 `apps/web/src/shared/api/apiToken.ts`，完整内容：

```typescript
const STORAGE_KEY = 'memory_anki_api_token'
const URL_PARAM = 'api_token'

export function getApiToken(): string {
  try {
    return localStorage.getItem(STORAGE_KEY) ?? ''
  } catch {
    return ''
  }
}

export function setApiToken(token: string) {
  try {
    if (token) localStorage.setItem(STORAGE_KEY, token)
    else localStorage.removeItem(STORAGE_KEY)
  } catch {
    // localStorage 不可用时静默降级（本机直连场景不需要令牌）。
  }
}

/** 首访支持 https://<tailscale-host>/?api_token=xxx 一次性写入并清理地址栏。 */
export function initApiTokenFromUrl() {
  if (typeof window === 'undefined') return
  const url = new URL(window.location.href)
  const token = url.searchParams.get(URL_PARAM)
  if (!token) return
  setApiToken(token)
  url.searchParams.delete(URL_PARAM)
  window.history.replaceState(null, '', url.toString())
}
```

**自查点**：`cd apps/web && npm run typecheck` 通过。

### 步骤 8：后端测试

新建 `apps/api/tests/test_api_token_auth.py`：

```python
import importlib

from fastapi import FastAPI
from fastapi.testclient import TestClient

from memory_anki.core.api_token_auth import ApiTokenAuthMiddleware


def build_app(token):
    app = FastAPI()

    @app.get("/api/v1/ping")
    def ping():
        return {"ok": True}

    @app.get("/index.html")
    def web():
        return {"web": True}

    app.add_middleware(ApiTokenAuthMiddleware, token=token)
    return app


def test_no_token_configured_allows_everything():
    client = TestClient(build_app(None))
    assert client.get("/api/v1/ping").status_code == 200


def test_remote_request_without_token_rejected():
    client = TestClient(build_app("secret"))
    resp = client.get("/api/v1/ping", headers={"X-Forwarded-For": "100.64.0.9"})
    assert resp.status_code == 401


def test_remote_request_with_token_allowed():
    client = TestClient(build_app("secret"))
    for headers in (
        {"X-Forwarded-For": "100.64.0.9", "X-Memory-Anki-Token": "secret"},
        {"X-Forwarded-For": "100.64.0.9", "Authorization": "Bearer secret"},
    ):
        assert client.get("/api/v1/ping", headers=headers).status_code == 200


def test_web_static_paths_not_blocked():
    client = TestClient(build_app("secret"))
    resp = client.get("/index.html", headers={"X-Forwarded-For": "100.64.0.9"})
    assert resp.status_code == 200
```

说明：`TestClient` 默认 client host 为 `testclient`（非回环），配合 `X-Forwarded-For` 可稳定模拟远程请求。不要在既有测试文件里插入用例。

**自查点**：`cd apps/api && python -m pytest tests/test_api_token_auth.py -q` 全绿。

## 3. 测试验收标准

可执行命令：

- `cd apps/api && python -m pytest tests/test_api_token_auth.py -q` → 全部通过。
- `cd apps/api && python -m pytest` → 既有用例无新增失败（未设置环境变量时中间件全放行）。
- `cd apps/web && npm run typecheck && npm run test && npm run build` → 全部通过。

行为验收（操作 → 期望现象）：

1. `.env` 不配置令牌，启动服务，本机浏览器操作全部功能 → 与现状完全一致，无 401。
2. `.env` 配置 `MEMORY_ANKI_API_TOKEN=abc123` 并重启；本机 `curl http://127.0.0.1:8012/api/v1/runtime-health` → 200（回环豁免）。
3. 手机经 Tailscale HTTPS 直接访问接口 → 401；首次访问 `https://<host>/?api_token=abc123` 后刷新 → 页面正常加载数据（令牌已写入 localStorage 并随请求头发送）。
4. 手机访问 `https://<host>/api/attachments/<某文件名>` 不带令牌 → 401。

回归检查：

- PWA 静态资源（`/`、`/assets/*`、`/pwa-reset.html`）在配置令牌后仍可匿名加载。
- 离线突变队列（`mutationQueue`）重放请求仍携带令牌头，重放成功。
- `X-Request-ID` 响应头在 401 响应上仍存在（日志中间件在外层）。
