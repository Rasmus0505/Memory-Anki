# 09-09 i18n 文案抽离基建

## 1. 原始需求

## 2. 详细执行清单

1. 新增 `apps/web/src/shared/i18n/messages.ts`。
2. 新增 `apps/web/src/shared/i18n/messages.test.ts`。
3. 修改 `apps/web/src/app/shell/GlobalCommandPalette.tsx` 使用 `translateAppMessage`。
4. 默认 locale 保持 `zh-CN`，显式 `en-US` 可返回英文文案。
