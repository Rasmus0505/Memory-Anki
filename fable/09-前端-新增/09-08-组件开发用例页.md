# 09-08 组件开发用例页（/dev playground）

## 1. 原始需求

设计 token 和基础组件需要一个开发期可视化入口。目标新增 `/dev/tokens`，仅在 `import.meta.env.DEV` 下注册，避免影响生产路由。

## 2. 详细执行清单

1. 新增 `apps/web/src/app/dev/DevTokensPage.tsx`。
2. 在 `apps/web/src/app/router/appRoutes.tsx` 中用 `import.meta.env.DEV` 注册 `/dev/tokens`。
3. 使用 `lazyWithRetry` 加载开发页。

## 3. 测试验收标准

- `cd apps/web && npm run typecheck` 通过。
- `rg -n "DevTokensPage|/dev/tokens" apps/web/src/app` 能看到开发路由和页面。
- 回归要求：生产环境不暴露该路由。
